# 3 个让人停不下来的英文播客

如果你的英文听力已经过了基础阶段，想通过播客提升同时又享受内容，这 3 个节目非常推荐。

## 1. How I Built This

NPR 出品的商业类播客，每期采访一个知名品牌的创始人，讲述创业故事。

## 2. The Daily

《纽约时报》的每日新闻播客，每集 20 分钟，用故事化的方式讲新闻。

<PodcastCard
  title="The Daily"
  cover="https://picsum.photos/seed/thedaily/80/80"
  platform="apple"
  description="比看新闻更有深度，比长篇报道更轻松，通勤完美伴侣。"
  link="https://podcasts.apple.com/"
/>

## 3. Lex Fridman Podcast

超长对谈节目，嘉宾从科学家到企业家到哲学家，每集 2-4 小时。

<PodcastCard
  title="Lex Fridman Podcast"
  cover="https://picsum.photos/seed/lexfridman/80/80"
  platform="spotify"
  description="深度对谈的极致，适合长途旅行或周末午后。"
  link="https://open.spotify.com/"
/>

## 听英文播客的建议

1. 先从 20 分钟左右的节目开始，不要一上来就挑战 3 小时长对谈
2. 听不懂的地方不要反复倒退，继续往下听，上下文会帮你理解
3. 开启 1.2x 倍速，逐渐适应正常语速
