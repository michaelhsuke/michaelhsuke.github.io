# B站那些让我笑到不行的宝藏 UP 主

分享几个我的快乐源泉，心情不好的时候翻出来看看，瞬间治愈。

## 搞笑区

### 老番茄
B站知识区 + 搞笑区双栖大佬。他的大学生活系列、幻想系列每一集都是笑点密集又不失温情的佳作。

### 盗月社食遇记
美食探店 + 搞笑互动，每次看他吃东西都特别有食欲，而且回访小店的故事很暖心。

### 伊森
生活区 UP 主，视频风格真实又有趣，尤其是他测评各种奇奇怪怪商品的视频，笑到肚子疼。

## 生活区

### 才浅
手工大神，用最原始的工具做最精致的东西。看他复原三星堆面具那期，完全被震撼到了。

### 漫游者
旅行 Vlog 拍得很有电影感，配上他的旁白，每条视频都是一部微纪录片。
