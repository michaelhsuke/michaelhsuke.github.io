# YouTube 上让我学到很多的 5 个频道

YouTube 是我最大的学习资源之一。以下 5 个频道，让我在工作之外学到了很多有趣的东西。

<VideoEmbed platform="youtube" youtubeId="dQw4w9WgXcQ" />

## 1. Ali Abdaal

前英国医生转全职 YouTuber，专注生产力和学习方法论。他的「如何更聪明地学习」「如何管理时间」系列视频干货满满。

## 2. 3Blue1Brown

数学可视化天花板。用动画的方式讲解线性代数、微积分、神经网络，看他的视频才觉得数学可以这么美。

## 3. Veritasium

科学科普频道，每期深入浅出地讲一个科学现象。关于「电的真相」那期视频颠覆了我此前的认知。

## 4. Mark Rober

前 NASA 工程师，现在专做大型科学实验和工程玩具。用油漆弹打松鼠那段是我的年度最佳。

## 5. Fireship

程序员的最爱。「100 Seconds of Code」系列用 100 秒讲清楚一个编程概念，节奏快信息密度高。
